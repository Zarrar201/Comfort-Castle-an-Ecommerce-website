# Comfort-Castle-an-Ecommerce-website
Welcome to Comfort Castle, a modern and user-friendly e-commerce platform designed for selling home comfort products such as furniture, decor, bedding, and more. This project is built to provide a smooth shopping experience with responsive design and a powerful admin dashboard.


# Features

 Product Browsing: View detailed listings by category, price, or popularity.

 Shopping Cart & Checkout: Add, update, and remove items with real-time cart updates.

 Order Management: Users can track orders; admins can manage inventory and sales.

 Admin Dashboard: Manage products, categories, users, and orders.

 Responsive Design: Optimized for desktops, tablets, and mobile devices.
